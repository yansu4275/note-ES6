解构赋值，左右两边结构格式要保持一致

let [a, [b, c] ] = [1, [2, 3] ]

document.write(a, b ,c)

可以赋默认值，如果后面有数据，则以后面的为准

let [aa, bb, cc='暂无....'] = ['aa', 'bb', ]

document.write(aa, bb, cc)

交换数据位置

[aaa, bbb] = [bbb, aaa]

用结构赋值处理json

可以给json中的数据起个别名，但后面也展示也必须用别名

let {name, gender, job:job1} = { name:'严肃', gender:'男', job:'攻城狮' }

document.write(name, gender, job1)
